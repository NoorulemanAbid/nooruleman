**Integrated genomic analysis suite with 8 specialized modules:**



```mermaid

    A[Mining] --> A1[HMMer Search/Build]

    B[Classification] --> B1[AlignSeqPro]

    C[Evolutionary] --> C1[Homologous Pair Finder]

    D[Primer Design] --> D1[GenePrimer/X]
    E[Visualization] --> E1[Gene Density Map]
    E --> E2[Gene Expression Analyzer]
```
